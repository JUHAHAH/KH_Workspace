스프링 부트 프로젝트 실행 방법

		src/main/java 하위 패키지에
		xxxApplication.java 파일이 프로젝트 생성시 자동 실행

		해당 클래스에는 main 메서드도 존재
		실행 시 내장된 Tomcat 서버를 통해 실행

프로젝트 구조

  • src/main/java: 자바 소스코드 모음
  • src/main/resources: 자바 자원/설정파일 모음
    • templates: 자바 탬플릿(응답화면) 모음(.html + thymeleaf)
    • static: templates 에 적용될 정적 자원 모아두는 폴더(.js, .css, 이미지 등)
    • application.properties: 프로젝트 관련 전반적 설정 모음
  • Project and External Dependacies
  • build.gradle: 빌드 도구(Gradle) 관련 파일 모음
