서버 제출 테스트 1

        이름과 나이를 입력받아서 서버(Java)에 제출

        *준비물*

        1. form 태그: input 태그안의 값을 지정된 경로로 제출
            action: 제출 경로
            method: 제출 방법 (GET/POST) 기본값: GET

            제출 == 클라이언트가 서버로 요청

        2. input 태그: 값을 입력할 있도록 해줌
            name: 서버로 값을 제출할 때 Key 역할, 입력된 값, 또는 input 태그의 
            value: 서버로 제출 시 value 역할    

        GET: GET을 통한 요청은 url 주소 끝에 파라미터로 포함되어 전송됨 == query string
             URL 끝에 ?key=value&key2=value2...
             -> 서버에서 name1, name2 라는 파라미터 값을 전달받을 수 있다
        [장점]
        - 단순하다
        - 캐싱 가능 (북마크, 방문 기록, 주소공유등 가능)

        [단점]
        - 보낼 수 있는 글자 수 제한 있음
        - 보안에 취약
