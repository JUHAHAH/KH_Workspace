FORM 태그

            > 앞서 했던 div와 같은 형식(블록)
            > 내부에 입력된 정보를 서버에 전달하는 역할

            [속성]
            1> action: 내부에 작성된 값을 전달 할 서버/ 페이지 주소를 작성
            2> name: form 태그에 이름을 지정하여 구분 가능(JAVA로 확인 가능)
            3> method: 서버로 데이터 전달 방식 지정
            4> target: action에 지정된 주소를 현재 탭 혹은 새탭 중 어디서 열것인지 지정
