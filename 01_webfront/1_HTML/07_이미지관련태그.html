img 태그

        웹문서에 사진 그림과 같은 이미지 삽입 가능하다

        [속성]
        1) source 위치 파일이나 웹주소
        2) width/height 지정 가능
        3) alt 이미지 속성 설명: 이미지가 출력되지 않는 경우 화면에 출력하는 이름

width, height 값은 고정 단위와 가변 단위로 정의

원본 사이즈 비율

고정 단위 (px): 외부 변화 상관 없이 동일 크기 유지

가변 단위 (%): 화면이나 영역 기준, 화면에 크기에 맞춰 자동 조절 가능

alt 속성 확인
