CSS(Cascading Style Sheet)

            마크업 언어(.html)가 실제 화면에
            표시되는 방법(모양/색상등의 스타일)을 기술하는 언어
            (.html이 어떤식으로 보여지는 지 지정)

CSS Selector

            Selector란 현재 HTML 문서 내에서 특정한 요소를 선택하는 문자/문자열
            스타일을 적용하고자 하는 요소이자 방법

태그 선택자

            목표 HTML과 같은 태그를 지정하는 선택자

            ex) 태그 {
                적용 스타일;
            }

ID 선택자

            HTML 문서내에서 id값이 일치하는 요소를 선택
            
            #id값 {
                적용 스타일;
            }

Class 선택자

            html내의 같은 class 속성을 지니는 모든 요소를 선택(그룹화)
            
            .class 속성값 {
                적용 스타일;
            }

모든 요소 선택자 (*)

            말 그대로 모든 요소를 선택하는 선택자
            
            * {
                적용 스타일;
            }
