스크립트 언어란?

        기본 프로그램의 동작을
        사용자의 요구에 맞게 수행하게 해줌

        별도의 소스코드 컴파일 과정이 없다
        Interpreter를 이용하여 코드를 바로 실행

        > 컴파일: 컴퓨터가 알아들을 수 있는 언어로 번역
        > Interpreter: 한줄씩 해석하여 실행하는 기계

        장점:
        > 컴파일에 소요되는 시간이 없음

        단점:
        > 줄로 읽기 때문에 상대적으로 프로그램 실행 시간이 느림

자바스크립트란?

        웹브라우저에서 주로 사용하는 인터프리터 방식의 
        '객체 지향 프로그래밍 언어'

        객체 지향 프로그래밍 언어: 객체끼리 연결하여 명령을 수행(그룹화)

자바스크립트 코드 작성법

  1. 브라우저 개발자 도구 -> 브라우저 콘솔에 직접 작성
  2. .html 내부의 script 태그를 이용하여 작성(Internal 방식)

  3. HTML 외부에 존재하는 .js 파일을 연결해서 사용하는 방법(External 방법)
  4. 태그에 직접 js 코드를 작성하는 방법(Inline 방식)
