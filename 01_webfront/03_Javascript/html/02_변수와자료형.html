변수(Variable)

        메모리에 값을 바꿔가며 저장할 수 있는 공간
        이름을 붙이고 값을 대입하면 변수 지정 가능

        
변수 명명 규칙

        > 의미있는 이름을 작성(한눈에 알아볼 수 있는 이름)
        > 대,소문자 구분
        > 특수 문자 '_'만 사용 가능(단, 첫 글자 제외)
        > 숫자도 사용가능(단 첫 글자 제외)
        > 카멜 표기법으로 작성(Camel Case): Hello World -> helloWorld

Javascript 변수의 종류와 특징

        var: 변수, 변수명 중복 가능(덮어쓰기 가능), 함수레벨 scope(함수내에서만 존재)
        let: 변수, 변수명 중복 안됨, 블록{} 레벨 scope
        const: 상수, 변수명 중복 안됨, 블록{} 레벨 scope, 절대값: 변화 불가

        1순위 const: JS는 특정 요소를 선택하여 사용하는 경우가 많기 때문에 const 지정 많이 함수내에서만
        2순위 let
        3순위 var

자바스크립트의 자료형

        자바스크립트는 변수 선언 시 별도의 자료형을 지정하지 않음
        > 변수에 대입되는 값(literal)에 의해서 자료형이 결정됨(int, str 구분)

        > string: 문자역
        > number: 정수/실수 모두 포함
        > boolean: 논리값 (true/false 혹은 1/0)
        > object: 객체 -> array(배열), {k:v, k:v, ...}자바스크립트 객체
        > functon: 함수
        > undefined: 정의되지 않은 변수(선언만 되고 값이 없음)

        + null: 정의할 수 없는(참조가 없는) 값
